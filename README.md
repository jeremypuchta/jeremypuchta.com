# jeremypuchta.com

My personal portfolio built with <a href="https://nextjs.org/">Next.js</a>, <a href="https://reactjs.org/">React</a>, 
<a href="https://mdxjs.com/">MDX</a> and <a href="https://tailwindcss.com/">Tailwind CSS</a>. It's built and deployed 
with <a href="https://vercel.com/">Vercel</a>.

This is my home base on the wonderful world wide web. Currently I use it solely as a blog to share
things I've learned and my thoughts about various subjects.

- <a href="https://nextjs.org/">Next.js</a>
- <a href="https://reactjs.org/">React</a>

- <a href="https://tailwindcss.com/">Tailwind CSS</a>
